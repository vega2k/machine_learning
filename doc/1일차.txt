해당 cell 실행하는 단축키
ctrl + enter : 해당 셀만 실행
alt + enter : 해당 셀을 실행하고, 새로운 셀을 추가
shift + enter : 해당 셀을 실행하고, 커서를 아래 셀에 

셀의 유형이 code와 markdown
esc + m : markdown
esc + y : code

code 주석  : ctrl + / 

code 자동완성 : tab (잘 않되면 tab 두번)
api 문서보기 : shift + tab
